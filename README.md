# About
ASP.NET SignalR server chat solution NuGet package.

Self-host chatting solution with cross-platform client application.

## Supported Platforms
- Windows 7 or higher
- macOS 11 or higher
- Android 7.0 (API 24) or higher
- iOS 14 or higher
